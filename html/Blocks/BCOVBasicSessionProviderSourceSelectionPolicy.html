BCOVBasicSessionProviderSourceSelectionPolicy Block Reference

Declared in BCOVBasicSessionProvider.h

Block Definition

BCOVBasicSessionProviderSourceSelectionPolicy

Each time a new video is ready to be loaded into a playback session, this block is called and the video is passed in. The block must return a source, which will be used to configure the AVPlayer of the playback session. If the block returns nil, the session provider will not deliver this session and will move on to the next session.

typedef BCOVSource* (^BCOVBasicSessionProviderSourceSelectionPolicy) (BCOVVideo *video)
